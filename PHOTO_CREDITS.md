# Photo credits

The trip page uses a fixed, manually reviewed set of locally stored photographs. Runtime image search is intentionally disabled so that every photo remains relevant, stable, and available offline after the page has loaded.

Most photographs originate from Flickr and were discovered through Openverse. The individual source page and license are linked directly from the image lightbox. One Monemvasia photograph comes from Pexels, and the Odontotos photograph comes from Wikimedia Commons.

The local copies are resized for web delivery. Copyright and license ownership remain with the original creators.
